# Shubham Pardule — Portfolio

Personal portfolio website built with Astro, Tailwind CSS v4, and TypeScript.

## Features

- Interactive particle hero background
- Physics-based tech stack chips (Matter.js)
- Asteroid Dodge mini-game
- Spring cursor trail effect
- Dark/light mode toggle with circular reveal effect
- Smooth scroll animations
- Contact form via Formspree
- Fully responsive design

## Getting Started

```sh
corepack pnpm install
corepack pnpm dev
```

| Command        | Action                                      |
| :------------- | :------------------------------------------ |
| `corepack pnpm install` | Install dependencies                        |
| `corepack pnpm dev`     | Start local dev server (usually `localhost:4321`) |
| `corepack pnpm build`   | Build production site to `./dist/`           |
| `corepack pnpm preview` | Preview build locally before deploying       |
